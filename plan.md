# 金融机构采购平台清单收集计划

## 目标
收集所有银行、理财子公司、信托公司、证券公司、基金公司、AIC公司、AMC公司的官方网站或其招标/采购平台，形成完整清单。

## 分类体系

### Stage 1 — 并行搜索（7路并发）

| 工作组 | 负责范围 | 搜索重点 |
|--------|---------|---------|
| 银行组_国有大行 | 6家国有大行 + 3家政策性银行 | 总行集中采购平台、招标公告栏目 |
| 银行组_股份制 | 12家全国性股份制商业银行 | 官网招标/采购栏目、集采平台 |
| 银行组_城农商 | 主要城商行、农商行、农信社 | 官网公告、采购平台 |
| 理财子公司组 | 约30家银行理财子公司 | 官网采购公告、招标信息 |
| 信托证券组 | 68家信托公司 + 主要证券公司 | 官网招标、采购平台 |
| 基金AMC组 | 主要基金公司 + 5大AMC | 官网采购、招标公告 |
| AIC组 | 5家金融资产投资公司 | 官网、母公司采购平台 |

### Stage 2 — 验证整合
- 去重、验证URL可访问性
- 按机构类型分类整理
- 标注平台类型（自有平台/第三方平台/官网栏目）

### Stage 3 — 输出交付
- Excel清单表格
- Markdown清单文档
- 纳入 tender-daily-dispatch SKILL.md 作为搜索源

## 输出格式

| 字段 | 说明 |
|------|------|
| 机构类型 | 银行/理财子/信托/证券/基金/AIC/AMC |
| 机构名称 | 全称 |
| 机构简称 | 常用简称 |
| 采购平台URL | 可直接访问的招标/采购页面 |
| 平台类型 | 自有集采平台/官网栏目/第三方代理 |
| 备注 | 访问方式、注册要求等 |

## 技能加载
- Stage 1: `deep-research-swarm`（并行搜索）
- Stage 3: `xlsx`（Excel输出）
